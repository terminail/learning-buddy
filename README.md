# Programming Learning VSCode Buddy

A collection of programming learning extensions for VSCode.

## C++ Primer 5th Edition Buddy

This is a VS Code extension that helps C++ Primer 5th Edition learners navigate their project files with a directory tree view.

For detailed information about this extension, please see [cpp-primer-buddy/README.md](cpp-primer-buddy/README.md).
