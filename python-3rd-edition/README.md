# Python Crash Course 3rd Edition Buddy

A Visual Studio Code extension that helps you navigate the Python Crash Course 3rd Edition course content directly from the sidebar, making it easier to learn Python through hands-on practice.

## About Python Crash Course 3rd Edition

**Python Crash Course** by Eric Matthes is a comprehensive guide to learning Python programming. The book covers fundamental programming concepts through carefully designed exercises and projects that help you write clean, readable code and understand Python's powerful features.

This book takes you from the basics of programming to more advanced topics like web applications, data visualization, and testing. Through hands-on projects including a space invaders-style game, data visualization with matplotlib, and a web application, you'll learn to think like a programmer and apply your skills in real-world scenarios.

The book emphasizes practical learning with exercises that encourage active coding and problem-solving. Each chapter builds upon previous concepts, gradually increasing in complexity to ensure a solid foundation in Python programming.

This book is an excellent resource for beginners who want to learn Python programming through practical, hands-on exercises and projects.

## How This Extension Helps You Learn

This extension makes it easier to navigate and work with the comprehensive exercises for Python Crash Course 3rd Edition, organized by chapter and topic. All exercises follow Python best practices and the pedagogical approach of providing conceptual guidance in TODO comments rather than code solutions, encouraging active learning through implementation.

With this extension, you can:
- Easily browse through all chapters of exercises directly in VS Code
- Navigate to specific exercises and example code quickly
- Open exercise files directly in the editor for hands-on practice
- Follow the structured learning path from basic concepts to advanced topics

## Features

- Displays the complete directory structure of the Python Crash Course 3rd Edition content in a tree view
- Easy navigation through chapters, exercises, and example code
- Direct file opening from the tree view
- Refresh functionality to update the view when content changes
- Numerically sorted chapters for logical progression

## Exercise Structure

Each exercise follows a consistent template structure:
- `README.md` - Detailed instructions and learning objectives
- `.py` files - Exercise and solution source files
- `test_*.py` - Test files for verifying solutions
- `pyproject.toml` - Project configuration
- `run_tests.py` - Test execution script

## Usage

1. Install the extension
2. Look for the "Python Crash Course 3rd Edition Buddy" icon in the VS Code activity bar
3. Click on the icon to open the course content tree view
4. Browse through the chapters and files
5. Click on any file to open it directly in the editor
