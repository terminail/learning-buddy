# License Instructions

## For Development and Testing

A test license has been generated and saved to `test-license.txt` in this directory.

### To use the test license:

1. Open Visual Studio Code
2. Install the extension (if not already installed):
   - Go to Extensions (Ctrl+Shift+X)
   - Click on the "..." menu and select "Install from VSIX"
   - Select the `cpp-primer-buddy-0.0.1.vsix` file
3. Open the License Manager:
   - Open the C++ Primer Buddy panel
   - Click on the "License Manager" button
4. Use the test license:
   - Open `test-license.txt` and copy its contents
   - Paste the license key into the license input field in the License Manager
   - Click "Add License"

### To generate a new test license:

Run the following command in your terminal:

```
node create-test-license.js
```

This will generate a new license and save it to `test-license.txt`, overwriting the previous one.

## For End Users

End users should:

1. Scan the WeChat QR code in the extension
2. Contact the instructor to purchase access
3. Receive their license key via WeChat
4. Enter their license key in the License Manager
