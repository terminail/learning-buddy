# Exercise: Sorting Algorithms

This exercise focuses on sorting algorithms in C++ STL.

## Learning Objectives

- Use std::sort with default ordering
- Work with std::sort and custom comparators
- Use std::stable_sort to maintain relative order of equal elements
- Work with partial sorting algorithms (std::partial_sort, std::nth_element)
- Understand performance implications of different sorting approaches
