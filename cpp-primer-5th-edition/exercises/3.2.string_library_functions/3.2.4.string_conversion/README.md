# Exercise: String Conversion

This exercise focuses on string conversion functions in C++.

## Learning Objectives

- Use std::to_string() to convert numbers to strings
- Work with std::stoi(), std::stol(), std::stoll() to convert strings to integers
- Use std::stof(), std::stod(), std::stold() to convert strings to floating-point numbers
- Handle conversion exceptions properly
- Work with std::stringstream for complex parsing
