# Exercise: String Searching

This exercise focuses on string searching functions in C++.

## Learning Objectives

- Use find() to search for substrings in strings
- Work with rfind() to search backwards
- Use find_first_of() and find_last_of() for character set searching
- Use find_first_not_of() and find_last_not_of() for inverse searching
- Handle search results and npos values properly
