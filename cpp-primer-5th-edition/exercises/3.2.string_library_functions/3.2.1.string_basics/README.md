# Exercise: String Basics

This exercise focuses on string creation, initialization, and basic operations in C++.

## Learning Objectives

- Create strings using different initialization methods
- Work with string capacity and size functions
- Access individual characters in strings
- Perform basic string operations and assignments
- Understand string equality and basic comparison

## Files

- [src/string_basics_exercises.cpp](src/string_basics_exercises.cpp) - Exercise template with TODO comments
- [src/string_basics_solutions.cpp](src/string_basics_solutions.cpp) - Complete solution implementation
- [test/test_string_basics_basics.cpp](test/test_string_basics_basics.cpp) - Basic unit tests
- [test/test_string_basics_practice.cpp](test/test_string_basics_practice.cpp) - Additional practice tests
