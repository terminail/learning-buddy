# Chapter 16.1b: Class Templates

This directory contains exercises for learning about class templates in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Class Templates
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to define and instantiate class templates
- How to use multiple template parameters
- How to provide default template parameters
- How class templates interact with inheritance
- Best practices for designing generic classes

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Classes and objects
- Function templates
- STL containers (basic understanding)

# Build the exercise
