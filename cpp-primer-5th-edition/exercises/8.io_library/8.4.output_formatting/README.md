# Exercise 8.1d: Output Formatting

This directory contains exercises for learning about output formatting in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Output Formatting
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to use stream manipulators for formatting
- How to control field width and fill characters
- How to format numeric values with precision
- How to use different number bases (decimal, hexadecimal, octal)
- How to create formatted tables and reports
- Best practices for professional output formatting

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Fundamental IO operations (std::cin, std::cout)
- Basic understanding of streams
- Basic formatting concepts
