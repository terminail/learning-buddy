# Exercise 8.1c: File Streams

This directory contains exercises for learning about file streams in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### File Streams
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to open files for reading and writing
- How to read data from text files line by line
- How to write data to text files
- How to work with binary files
- How to handle file opening errors
- Best practices for file I/O operations

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Fundamental IO operations (std::cin, std::cout)
- Basic understanding of streams
- File system concepts
