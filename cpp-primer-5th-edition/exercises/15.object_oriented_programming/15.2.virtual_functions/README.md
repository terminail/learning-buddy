# Chapter 15.1b: Virtual Functions

This directory contains exercises for learning about virtual functions and polymorphism in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Virtual Functions
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to declare and implement virtual functions
- The difference between static binding and dynamic binding
- How virtual functions enable polymorphism
- How to create abstract base classes with pure virtual functions
- The importance of virtual destructors in inheritance hierarchies

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Classes and objects
- Inheritance concepts
- Member functions and access specifiers
