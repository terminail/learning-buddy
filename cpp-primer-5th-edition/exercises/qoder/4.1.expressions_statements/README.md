# Chapter 4.1: Expressions and Statements

This directory contains exercises for learning about expressions and statements in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### 4.1a: Conditional Expressions
**Directory**: [conditional_expressions](conditional_expressions/)
Introduction to conditional expressions and statements including ternary operators, if-else statements, and switch statements.

## Learning Progression

Complete the exercises in numerical order:
1. [conditional_expressions](conditional_expressions/) - Fundamental conditional expressions and statements

## Getting Started

Each exercise directory contains:
- `src/` - Source code files (exercise and solution)
- `test/` - Unit tests
- `CMakeLists.txt` - Build configuration
- `README.md` - Exercise-specific documentation
- `build.sh` - Build script for Linux/Docker
- `test_runner.sh` - Test execution script
