## Key Concepts Covered

### String Searching
- **find()**: Locate first occurrence of substring
- **rfind()**: Locate last occurrence of substring
- **npos**: Special value indicating "not found"

### Substring Operations
- **substr()**: Extract part of a string
- **Exception handling**: Handle out-of-bounds substring operations

### String Comparison
- **Operators**: ==, !=, <, >, <=, >= for lexicographical comparison
- **compare()**: Detailed comparison with return values

### String Modification
- **replace()**: Substitute parts of strings
- **erase()**: Remove parts of strings

### String Conversion
- **to_string()**: Convert numbers to strings
- **stoi(), stof()**: Convert strings to numbers
- **Error handling**: Handle conversion exceptions
