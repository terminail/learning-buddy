# Chapter 3.2: String Library Functions

This directory contains exercises for learning about string library functions in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### 3.2a: String Operations
**Directory**: [string_operations](string_operations/)
Introduction to common string library functions including searching, substring extraction, and comparison.

## Learning Progression

Complete the exercises in numerical order:
1. [string_operations](string_operations/) - Fundamental string library functions

## Getting Started

Each exercise directory contains:
- `src/` - Source code files (exercise and solution)
- `test/` - Unit tests
- `CMakeLists.txt` - Build configuration
- `README.md` - Exercise-specific documentation
- `build.sh` - Build script for Linux/Docker
- `test_runner.sh` - Test execution script
