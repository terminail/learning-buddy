# Chapter 17.1b: Random Number Generation

This directory contains exercises for learning about random number generation in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Random Number Generation
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to create and use random number generators
- How to use different random number distributions
- How to seed generators for reproducible results
- How to generate random numbers in specific ranges
- Best practices for random number generation in C++

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Basic understanding of probability and statistics
- STL containers (basic understanding)
