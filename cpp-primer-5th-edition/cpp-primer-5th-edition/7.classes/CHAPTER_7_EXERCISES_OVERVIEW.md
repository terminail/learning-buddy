# Chapter 7: Classes Exercises Overview

This directory contains exercises for Chapter 7 of C++ Primer 5th Edition, which introduces classes in C++.

## Exercise Progression

### 7.1a: Class Basics
**Directory**: [class_basics](class_basics/)
**Focus**: Introduction to class definition and usage
- Class declaration and definition
- Member variables and member functions
- Access specifiers (public, private)
- Creating and using class objects

### 7.1b: Constructors Basics
**Directory**: [constructors_basics](constructors_basics/)
**Focus**: Understanding and implementing class constructors
- Default constructors
- Parameterized constructors and overloading
- Constructor initializer lists
- Copy constructors

### 7.1c: Destructors Basics
**Directory**: [destructors_basics](destructors_basics/)
**Focus**: Understanding and implementing class destructors
- Basic destructor implementation
- Resource management in destructors
- Destructor order and inheritance
- Exception safety in destructors

### 7.1d: Access Control
**Directory**: [access_control](access_control/)
**Focus**: Understanding access specifiers and encapsulation
- Public, private, and protected members
- Data encapsulation and information hiding
- Friend functions and classes
- Inheritance and access control

## Key Concepts Covered

### Class Fundamentals
- **Class Definition**: struct vs class keywords
- **Member Functions**: Defining methods inside and outside the class
- **Access Control**: Public, private, and protected members
- **Object Creation**: Instantiating class objects

### Constructors and Destructors
- **Default Constructor**: Automatic and explicit definition
- **Parameterized Constructors**: Flexible object initialization
- **Copy Constructor**: Creating objects from existing objects
- **Destructor**: Resource cleanup and object destruction

### Basic OOP Concepts
- **Encapsulation**: Bundling data and functions together
- **Data Hiding**: Controlling access to class members
- **Interface**: Public methods as the class interface

## Prerequisites

Students should be familiar with:
- Basic C++ syntax and data types
- Functions and function overloading
- Variables and scope
- Basic understanding of structs
