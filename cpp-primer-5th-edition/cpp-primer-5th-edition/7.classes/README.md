# Chapter 7.1: Classes

This directory contains exercises for learning about classes in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### 7.1a: Class Basics
**Directory**: [class_basics](class_basics/)
Introduction to classes including class definition, data members, and member functions.

## Learning Progression

Complete the exercises in numerical order:
1. [class_basics](class_basics/) - Fundamental class concepts

## Getting Started

Each exercise directory contains:
- `src/` - Source code files (exercise and solution)
- `test/` - Unit tests
- `CMakeLists.txt` - Build configuration
- `README.md` - Exercise-specific documentation
- `build.sh` - Build script for Linux/Docker
- `test_runner.sh` - Test execution script
