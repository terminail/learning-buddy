# Chapter 19.1b: Multiple Inheritance

This directory contains exercises for learning about multiple inheritance in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Multiple Inheritance
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to implement basic multiple inheritance
- How to handle the diamond problem with virtual inheritance
- How to use multiple inheritance for mixins and interface combination
- The differences between multiple inheritance and composition
- Best practices for using multiple inheritance in C++

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Inheritance and polymorphism
- Virtual functions
- Interface design principles
