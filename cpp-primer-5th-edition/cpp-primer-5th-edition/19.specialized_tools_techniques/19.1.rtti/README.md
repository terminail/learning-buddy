# Chapter 19.1a: Run-Time Type Identification (RTTI)

This directory contains exercises for learning about Run-Time Type Identification (RTTI) in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### RTTI
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to use the typeid operator for type identification
- How to use dynamic_cast for safe downcasting
- How to compare type information at runtime
- How to implement type-safe containers using RTTI
- Performance implications of RTTI
- Best practices for using RTTI in C++

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Inheritance and polymorphism
- Virtual functions
- Pointers and references
