# Chapter 18.1a: Namespaces

This directory contains exercises for learning about namespaces in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Namespaces
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to create and use namespaces to organize code
- How to create nested namespaces
- How to use namespace aliases for convenience
- How to resolve naming conflicts with scope resolution
- How to use using declarations and using directives
- Best practices for namespace design in large programs

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Functions and classes
- Scope rules in C++
