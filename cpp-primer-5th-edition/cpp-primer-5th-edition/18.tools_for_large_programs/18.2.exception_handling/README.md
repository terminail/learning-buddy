# Chapter 18.1b: Exception Handling

This directory contains exercises for learning about exception handling in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Exception Handling
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to use try-catch blocks for exception handling
- How to throw and catch standard exceptions
- How to create custom exception classes
- How to use exception specifications (noexcept)
- Exception safety guarantees (basic, strong, no-throw)
- Best practices for exception handling in large programs

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Functions and classes
- Inheritance and polymorphism
- Standard library exceptions
