# Exercise: Searching Algorithms

This exercise focuses on searching algorithms in C++ STL.

## Learning Objectives

- Use std::find and std::find_if for linear searching
- Work with std::count and std::count_if for counting elements
- Use std::binary_search and related functions for efficient searching
- Work with string searching algorithms
- Handle search results and edge cases properly

## Files

- [src/searching_algorithms_exercises.cpp](src/searching_algorithms_exercises.cpp) - Exercise template with TODO comments
- [src/searching_algorithms_solutions.cpp](src/searching_algorithms_solutions.cpp) - Complete solution implementation
