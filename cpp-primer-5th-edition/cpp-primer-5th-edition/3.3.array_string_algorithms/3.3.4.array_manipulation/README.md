# Exercise: Array Manipulation Algorithms

This exercise focuses on array manipulation algorithms in C++ STL.

## Learning Objectives

- Use std::reverse and std::reverse_copy to reverse element order
- Work with std::unique and std::unique_copy to remove consecutive duplicates
- Use std::remove and std::remove_if to mark elements for removal
- Work with rotation and shuffling algorithms
- Understand the erase-remove and erase-unique idioms
