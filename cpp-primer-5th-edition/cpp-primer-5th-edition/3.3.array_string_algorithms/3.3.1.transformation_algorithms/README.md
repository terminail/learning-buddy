# Exercise: Transformation Algorithms

This exercise focuses on transformation algorithms in C++ STL.

## Learning Objectives

- Use std::transform with function pointers
- Work with std::transform and lambda functions
- Use std::transform with custom functions and functors
- Perform string transformations with algorithms
- Understand different transformation interfaces

## Files

- [src/transformation_algorithms_exercises.cpp](src/transformation_algorithms_exercises.cpp) - Exercise template with TODO comments
- [src/transformation_algorithms_solutions.cpp](src/transformation_algorithms_solutions.cpp) - Complete solution implementation
- [test/test_transformation_algorithms_basics.cpp](test/test_transformation_algorithms_basics.cpp) - Basic unit tests
- [test/test_transformation_algorithms_practice.cpp](test/test_transformation_algorithms_practice.cpp) - Additional practice tests
