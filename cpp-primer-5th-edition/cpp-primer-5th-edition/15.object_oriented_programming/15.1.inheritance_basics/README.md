# Chapter 15.1a: Inheritance Basics

This directory contains exercises for learning about inheritance in C++ as covered in C++ Primer 5th Edition.

## Exercise Structure

### Inheritance Basics
**Directory**: [src/](src/)

## Learning Objectives

By completing these exercises, students will understand:
- How to create base classes and derived classes
- How inheritance promotes code reuse
- How to access base class members from derived classes
- The difference between public, protected, and private inheritance
- How to override base class functions in derived classes

## Prerequisites

Students should be familiar with:
- Basic C++ syntax
- Classes and objects
- Member functions and access specifiers
- Constructors and destructors
