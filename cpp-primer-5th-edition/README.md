# C++ Primer 5th Edition Buddy

A Visual Studio Code extension that helps you navigate the C++ Primer 5th Edition course content directly from the sidebar, making it easier to learn C++ through hands-on practice.

## How This Extension Helps You Learn

This extension makes it easier to navigate and work with the comprehensive exercises for C++ Primer 5th Edition, organized by chapter and topic. All exercises follow modern C++ practices and the pedagogical approach of providing conceptual guidance in TODO comments rather than code solutions, encouraging active learning through implementation.

With this extension, you can:
- Easily browse through all 20 chapters of exercises directly in VS Code
- Navigate to specific exercises and example code quickly
- Open exercise files directly in the editor for hands-on practice
- Follow the structured learning path from basic concepts to advanced topics

## Features

- Displays the complete directory structure of the C++ Primer 5th Edition content in a tree view
- Easy navigation through chapters, exercises, and example code
- Direct file opening from the tree view
- Refresh functionality to update the view when content changes
- Numerically sorted chapters for logical progression

## Exercise Structure

Each exercise follows a consistent template structure:
- `src/` - Contains exercise and solution source files
- `test/` - Contains basic and practice test files
- `CMakeLists.txt` - Build configuration
- `README.md` - Detailed instructions and learning objectives
- `build.sh` - Build script for Linux/Docker environments
- `run_tests.sh` - Test execution script

## Usage

1. Install the extension
2. Look for the "C++ Primer 5th Edition Buddy" icon in the VS Code activity bar
3. Click on the icon to open the course content tree view
4. Browse through the chapters and files
5. Click on any file to open it directly in the editor
